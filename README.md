## MicroTube

Lightweight Youtube app made with Solid, TypeScript and Vite

![Screenshot](./screenshot.png)

### API Installation

```
git clone https://github.com/kirpalmakanga/microtube-api
cd microtube-api
npm install
```

### Installation

```
npm install
```

### Start app

```
npm run dev
npm run dev:api
```
